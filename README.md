# &lt;cosmoz-sse&gt;

`<cosmoz-sse>` is an element that allows subscribing to server-sent events.

## Install

`npm install`

## Viewing Your Element

Start a web server @ port `8080` providing an event source at `/events`, then:

```
$ npm start
```

## Running Tests

```
$ npm test
```
